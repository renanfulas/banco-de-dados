# Projeto Banco Simples em Python

Este projeto simula um sistema bancário básico utilizando orientação a objetos em Python. Ele permite:

- Cadastro de cliente
- Consulta de saldo
- Depósito e saque
- Histórico de movimentações
- Persistência dos dados em arquivo

## Como usar

1. Execute o arquivo `Projeto 2.py`.
2. Utilize o menu interativo para realizar operações bancárias.
3. O histórico de movimentações é salvo em `saldo_cliente.txt`.

## Estrutura
- `Projeto 2.py`: Código principal do sistema.
- `Clientes.py`: Classe do cliente.
- `Conta.py`: Classe da conta bancária.
- `saldo_cliente.txt`: Histórico de movimentações.
